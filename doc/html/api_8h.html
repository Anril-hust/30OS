Detailed Description

OS api callback by INT 0x40.

Everytime application call os api. It actually trigger an interrupt with INT 0x40. OS use asm_os_console_api as the interrupt handler. you could refer it in globalTbl.c. API_xxx is called by asm_os_console_api according to edx. The routine array is listed in wnd_console.c named api_routine_array. Every API_xxx has an version named api_xxx. api_xxx is called by application.
