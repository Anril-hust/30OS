Detailed Description

basic functions related to sheet.

All drawing action operate on sheet. The sheet is similiar to Layer. It can hold pixels. All movement of action also based on sheet, The sheet is managed by SHEET_CTL. These file contains some utils functions used to operate the sheet. There are two ways to redraw the screen. 1st, directly call sheet_draw_sub(), sheet_draw_all() or sheet_draw_sub_withoutMouse(). That would be redrawn immediately. 2nd, call sheet_add_redraw_region_rect() or sheet_add_redraw_region() to add redraw request. That would be redrawn by sheet_daemon when CPU is idle.
