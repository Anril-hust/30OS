boolean wnd_create ( IN RECT  wndSize,
IN int8 *  title,
IN int8 *  icon,
OUT WND **  wnd,
draw_wnd  func 
)

create a window, with specified size, title, icon, draw function.

If create successful, the window would be put into sortedWnd[] of wndCtl.

Parameters
wndSizeRECT value of window size.
titlethe title of window, would display on title bar.
iconthe icon of window, would display on top left of window's title bar.
wndthe window handle, could be usd in future.
functhe draw function used to draw this window, this could be drawed to a default window or console. you could add your own draw function to draw your own window.
Returns
whether the operation is successful. if yes, return TRUE els FALSE.
