void interrupt_handler_0x20 ( int32 *  esp)

timer interrupt.

Add global timecounter. check sortedTimer[] in timerCtl to see if there are some timers which are timeout. Put the timeout timer to timerCtl.timeoutTimer[]. check sleep tasks list in taskCtl, if the task has sleep for enough time, wake up them and call scheduler.

Parameters
espstack pointer
Returns
NULL
